# AdPreview: View and Share Display Ads In A Single Browser Window
This repository provides a high-speed, zero-cost solution for hosting and previewing static and HTML5 display banners (Adobe Animate) at scale. It eliminates manual coding by using a custom Python indexer and GitHub Actions to build live client preview links in seconds.

🚀 How to Use

Prepare: Organize your banners into a main "Advertiser" folder (e.g., Tallgrass). Inside that folder, include your unzipped Adobe Animate folders (e.g., 300x250, 728x90).

Upload: Drag and drop the entire Advertiser folder directly into this GitHub repository via the browser.

Deploy: Click Commit changes. Wait 30–60 seconds for the GitHub Action to process the files (the yellow dot will turn into a green checkmark).

Share: The script automatically generates a master preview page inside your advertiser folder. Your unique, shareable link is:

https://irushabshah.github.io/adpreview/AdvertiserFolderName/.

🛠 Key Features

Dynamic Per-Client Indexing: Automatically creates a custom index.html for every advertiser folder uploaded, keeping projects organized and distinct.

Adobe Animate Integration: Automatically derives exact ad dimensions (width/height) directly from the HTML metadata generated by Adobe Animate.

Sync Preview: Includes a "Reload All Ads" button on every preview page to synchronize animations for precise client review.

Zero-Terminal Workflow: Designed for creative teams—no command prompt, Node.js, or local coding required.

Scalable Architecture: Supports unlimited unique client URLs within a single repository.
